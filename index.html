Mois 1 – Fondamentaux

Objectif : Acquérir les bases en mathématiques, physique et matériaux pour maîtriser les calculs de structure et la compréhension des forces.

Semaine 1 – Mathématiques appliquées

Théorie : Intégrales, dérivées, vecteurs et fonctions sont essentiels pour calculer les efforts et moments sur les structures. Les formules apprises ici seront utilisées dans les projets des mois 2 et 3.

Exemple explicatif :

Si une poutre subit une force variable F(x) = 2x + 3 kN sur 0 ≤ x ≤ 5 m, la charge totale :

  1. Calcul intégral : ∫(2x+3)dx = x² + 3x + C
  2. Évaluation : x=5 → 5²+15=40 kN·m
